# 📄 3. Markdown の記法：応用編  

## 表（Table）

### 書き方
```
| 見出し1 | 見出し2 |
|--------|--------|
| 内容A  | 内容B  |
| 内容C  | 内容D  |
```

### 表示例

| 見出し1 | 見出し2 |
|--------|--------|
| 内容A  | 内容B  |
| 内容C  | 内容D  |
