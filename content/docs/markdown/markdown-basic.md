# 📄 2. Markdown の記法：基本編  

## 見出し

### 書き方
```
# 見出し1
## 見出し2
### 見出し3
```

### 表示例
# 見出し1
## 見出し2
### 見出し3

---

## 太字

### 書き方
```
**太字**
```

### 表示例
**太字**

---

## リンク

### 書き方
```
[テキスト](リンク)
```

### 表示例
[Google](https://google.com)

---

## 画像リンク  


### 書き方
```
![説明](画像パス)
```

### 注意  
※実際には 閲覧ソフトによって若干書き方が異なる場合もある
