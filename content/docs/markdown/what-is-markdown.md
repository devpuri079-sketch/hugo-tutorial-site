## Markdown とは

Markdown は、プレーンテキストで書ける軽量マークアップ言語。  
文章を書く感覚で、見出し・太字・リンク・画像などを簡単に表現できる。

- 拡張子は `.md`
- GitHub、Hugo、Notion、Qiita など多くのサービスで採用
- 読みやすく、書きやすいのが特徴

---

## Markdown の閲覧方法

### ● GitHub で閲覧  
リポジトリに置くだけで自動でレンダリングされる。

### ● VS Code のプレビュー  
`Ctrl + Shift + V`（Windows）  
`Cmd + Shift + V`（Mac）

### ● 専用アプリ  
Typora、Obsidian、MarkText など。

### ● HTMLに変換して閲覧  
HugoなどのSSGを使用して、Markdown を HTML に変換して Web サイトとして表示できる。
