# Hugo のテーマを選定する

---

## この手順の目的

この手順では、Hugo で使用するテーマを選定するために、  
公式テーマ一覧ページを確認し、デモサイトやインストール手順を把握するところまでを行う。

---

## テーマ選定の手順

### 1. Hugo Themes のページを開く

Hugo 公式のテーマ一覧ページ（[Hugo Themes](https://themes.gohugo.io/)
）を開く。  
多数のテーマが並んでおり、デザインや用途に応じて選ぶことができる。

---

### 2. 気になるテーマのサムネイルをクリックする

一覧の中から気になるテーマを見つけ、サムネイルをクリックする。  
テーマの詳細ページが開く。

また、一覧ページには **Tags カテゴリー**があり、  
`blog`、`docs`、`portfolio` など用途別にテーマを絞り込むことができる。  
目的に合うテーマを効率よく探すために活用する。

---

### 3. Demo ボタンでサンプルサイトを確認する

テーマ詳細ページにある **Demo** ボタンをクリックし、  
そのテーマで構築されたサンプルサイトを確認する。  
レイアウト、色、フォント、トップページの構成などを確認し、  
自分のサイトの目的に合うか判断する。

---

### 4. Download ボタンで GitHub のページに遷移し、手順を確認する

テーマ詳細ページにある **Download** ボタンをクリックすると、  
テーマの GitHub リポジトリに遷移する。  
ここでは以下を確認する。

- インストール手順（submodule の追加方法など）  
- 必要な設定（config の追加項目）  
- 使用例  
- テーマ固有の注意点  
- チュートリアルやサンプルコード  

テーマによって設定方法が異なるため、導入前に必ず確認する。

---

### 5. GitHub 上のサンプルサイトのソースコードを確認する

多くのテーマは、GitHub リポジトリ内または関連リポジトリに  
**サンプルサイトのソースコード**を公開している。

- `exampleSite` フォルダ  
- `config.toml` のサンプル  
- ディレクトリ構成  
- 画像や記事の配置例  

これらを確認し、**ファイル構成や設定ファイルを模倣する**ことで、  
テーマを正しく動作させやすくなる。
