## このサイトについて
  社内向け研究発表として、ドキュメントの Web 化 と Hugo を用いた静的サイト構築 の調査内容をまとめたサイトです。<br/>
  従来のファイルベースのドキュメント管理から、Markdown＋SSG（Static Site Generator）による
  “軽量で検索しやすく、更新しやすいドキュメント管理” への移行を検討するための情報を整理しています。
{{% columns %}}
- ## ページ構成
  - Document ページ
  GitHub の設定手順、Hugo の構築手順など、実際に環境を作るための手順をまとめています。
  - Hugo の説明ページ
  ドキュメント管理の現状、Web 化の意義、Markdown→SSG の流れ、Hugo の特徴を解説しています。


- ## このサイトの使い方
  - まずは Hugo の説明ページ で背景と全体像を把握
  - 次に Document ページ で手順を確認しながら実際に環境構築
